# 달팽이질주 - AI 생성 영상 탐지 서비스

## 프로젝트 소개
CNN-LSTM 하이브리드 모델과 XAI 기술을 활용한 AI 생성 영상(딥페이크 및 T2V) 탐지 웹 서비스입니다.

## 기술 스택
- React + TypeScript
- Vite
- Zustand (상태관리)

## 시작하기

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

### 빌드
```bash
npm run build
```
